# 👨‍💼 Employee Attrition Analysis Dashboard

An AI-powered Employee Attrition Prediction and Analysis Dashboard built using Machine Learning and Streamlit.

## 🚀 Project Overview

This project helps organizations identify employees who are at high risk of leaving the company using Machine Learning algorithms.

The application provides:

- 📊 Employee Attrition Dashboard
- 🔮 Attrition Prediction System
- 📈 Feature Importance Analysis
- 🏅 Employee Satisfaction Insights
- ⚡ Real-time Prediction using Streamlit

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Pickle

---

## 📂 Project Features

### 📊 Dashboard
- Displays High-Risk Employees
- Shows Top Employee Satisfaction Data
- Feature Importance Visualization

### 🔮 Prediction Module
User can predict employee attrition based on:
- Age
- Monthly Income
- Overtime
- Job Satisfaction
- Distance From Home
- Department
- Gender

---

## 🧠 Machine Learning Model

The project uses a trained Machine Learning classification model to predict employee attrition probability.

### Model Workflow
1. Data Preprocessing
2. Feature Encoding
3. Feature Scaling
4. Model Training
5. Prediction & Probability Analysis
